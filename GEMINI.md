# AI 法律助手 — GEMINI.md (中国本地化版)

## 项目概述
**AI 法律助手** 是一套专为 **Claude Code** 打造的 14 个专业技能和 5 个并行 AI 代理。本项目已针对 **中华人民共和国法律体系**（特别是《民法典》、《个人信息保护法》、《劳动合同法》）进行了深度本地化。它旨在自动化合同审查、识别法律风险，并生成符合中国商业习惯的专业法律报告。

### 核心技术栈
- **Claude Code**: 技能与代理的执行环境。
- **Python 3.8+**: 驱动 PDF 报告生成引擎。
- **ReportLab**: 用于创建支持中文、视觉丰富的 PDF 法律报告。
- **Markdown**: 用于定义技能 (`SKILL.md`)、代理指令及报告模板。

---

## 项目结构 (本地化调整)

- `legal/SKILL.md`: **核心协调器**。已更新为中文指令，适配中国法律术语。
- `skills/`: 各项子功能实现。
  - `legal-review/`: 旗舰技能，启动 5 个针对中国法律优化的代理。
  - `legal-risks/`: 专门识别违约金超标 (30%)、竞业限制补偿缺失等中国法风险。
- `agents/`: 专门为中国法定制的 AI 代理。
  - `legal-risks.md`: 核心风险代理，基于《民法典》和司法解释。
  - `legal-compliance.md`: 侧重 PIPL (个人信息保护法) 和国内行业合规。
- `scripts/`: 后端工具脚本。
  - `generate_legal_pdf.py`: 已支持中文字体渲染 (SimHei/SimSun)。
- `templates/`: 符合中国公文习惯的报告模板。

## 核心命令

项目采用 `/legal <命令>` 结构。

### 旗舰命令：`/legal review <文件>`
启动 5 个代理进行深度审计：
1. **条款分析代理**：识别合同结构。
2. **风险评估代理**：基于中国法律进行 1-10 分评分。
3. **合规检查代理**：标记违反 PIPL、民法典强制性规定的项。
4. **期限映射代理**：梳理截止日期和法定义务。
5. **建议引擎代理**：提供中文替代文案。

## 开发与测试

### 安装
运行本地安装脚本以设置技能：
```bash
./install.sh
```
安装位置：
- **技能**: `$HOME/.claude/skills/`
- **代理**: `$HOME/.claude/agents/`

### PDF 生成环境
PDF 引擎需要 Python 和 `reportlab`。对于中文支持，系统需安装有 `SimHei.ttf` (黑体) 或 `SimSun.ttc` (宋体)。
```bash
pip3 install reportlab
```

## 设计规范
- **法律免责声明优先**：所有输出必须首先声明其 AI 生成属性，且不构成正式法律意见。
- **风险等级指示**：
  - 🔴 **高风险**: 严重不平等或违法违规，必须谈判。
  - 🟡 **中风险**: 存在偏向性，建议关注。
  - 🟢 **低风险**: 标准平衡条款。
- **中国法背景**：所有分析必须引用或遵循中国现行法律和司法解释。
- **成果产出**：报告以 Markdown 形式保存于当前目录，并可导出为 PDF。
